CREATE Branches & Tags Merge & Rebase
Clone an existing repository List all existing branches Merge < branch > into your current HEAD
$ git clone https://github.com/usernick/reponame $ git branch -av $ git merge < branch >
Create a new local repository Switch branch Rebase your current HEAD onto < branch >
$ git init $ git switch < branch > $ git rebase < branch >
Create a new branch Abort a rebase
LOCAL CHANGES $ git branch *isim* $ git rebase --abort
Changed files in your working directory Create a new tracking branch Continue a rebase after resolving conflicts
$ git status $ git checkout --track < branch > $ git rebase --continue
Changes to tracked files Delete a local branch Use your configured merge tool to solve conflicts
$ git diff $ git branch -d < branch > $ git mergetool
Add all current changes to the next commit Mark the current commit with a tag Use your editor to manually solve conflicts and (after resolving) mark file as resolved
$ git add . $ git tag < tag-name > $ git add < resolved-file >
Add some changes in < file > to the next commit $ git rm < resolved-file >
Commit all local changes in tracked files UPDATE & PUBLISH
$ git commit -a List all currently configured remotes UNDO
Commit previously staged changes $ git remote -v Discard all local changes in your working directory
$ git commit Show information about a remote $ git reset --hard HEAD
Change the last commit $ git remote show < remote > Discard local changes in a specific file
$ git commit --amend Add new remote repository, named < remote > $ git checkout HEAD < file >
$ git renıte add < shortname >< url > Revert a commit
COMMIT HISTORY Download all changes from < remote >, but don't integrate into HEAD $ git revert < commit >
Show all commits, starting with newest $ git fetch < remote > Reset your HEAD pointer to a previous commit ...and discard all changes since then
$ git log Download changes and directly merge into HEAD $ git reset --hard < commit >
Show changes over time for a spesific file $ git pull < remote >< branch > ...and preserve all changes as unstaged changes
$ git log -p < file > Publish local changes on a remote $ git reset < commit >
Who changed what and when in < file > $ git push < remote >< branch > ...and preserve uncommitted local changes
$ git blame < file > Delete a branch on the remote $ git reset --keep < commit >
$ git branch -dr < remote/branch >
Publish your tags
$ git push --tags
